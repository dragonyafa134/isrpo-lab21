## Описание
В ходе выполнения данной лабораторной работы было создано простое веб-приложение "Список задач" (TODO-лист). Проект направлен на закрепление навыков работы с DOM (Document Object Model), обработкой событий, а также на практическое применение HTML, CSS и JavaScript для создания динамического интерфейса.

### В ходе работы были изучены и применены следующие темы:
Работа с DOM:

* Поиск элементов на странице с помощью getElementById и querySelector.

* Создание новых элементов (createElement) и встраивание их в структуру документа.

* Динамическое обновление содержимого элементов (textContent).

Обработка событий:

* Навешивание обработчиков на кнопки (клик) и поля ввода (нажатие Enter).

* Реализация логики добавления, отметки о выполнении и удаления задач.
